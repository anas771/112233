# 🐔 نظام إدارة عنابر الدجاج اللاحم (Poultry Farm Management)
### النسخة المطورة v3.8 — نسخة احترافية ذكية تعمل بنظام الويندوز

منظومة متكاملة وسهلة الاستخدام تهدف إلى أتمتة حسابات وتكاليف مزارع الدواجن، مع دعم كامل للغة العربية والرسوم البيانية وتقارير PDF/Excel.

---

## 🚀 طريقة التشغيل (الأسهل والأسرع)

لقد قمنا بتطوير **المشغل الذكي** لجعل تشغيل النظام سهلاً جداً دون الحاجة لتثبيت مكتبات برمجية يدوياً.

1.  تأكد من وجود **Python** مثبت على جهازك.
2.  قم بتحميل ملفات المشروع في مجلد واحد.
3.  انقر نقراً مزدوجاً على ملف **`run.bat`**.

### ماذا يفعل ملف `run.bat`؟
*   يقوم بإنشاء بيئة برمجية معزولة تلقائياً.
*   يقوم بتثبيت كافة المكتبات المطلوبة (مثل `openpyxl`, `fpdf2`, `matplotlib`) في الخلفية.
*   يفتح لك النظام مباشرة.
*   **خيار موجه الأوامر**: سيسألك المشغل إذا كنت تريد إظهار "شاشة الأوامر السوداء" لمتابعة العمليات تقنياً أو تشغيل النظام بصمت.

---

## 🛠️ وظائف النظام الأساسية
*   **إدارة الدفعات**: إدخال دورات التسمين مع كافة تفاصيل التكاليف (كتاكيت، علف، أدوية، إيجار، إلخ).
*   **الحسابات التلقائية**: حساب EPEF (مؤشر الكفاءة الأوروبي)، FCR (معدل التحويل الغذائي)، وصافي الأرباح.
*   **الرسوم البيانية (Dashboard)**: لوحة قياس تفاعلية توضح اتجاهات الأرباح ونسب النافق.
*   **استيراد ذكي**: إمكانية استيراد دفعات كاملة من ملفات Excel أو حتى استيراد مجلد كامل دفعة واحدة.
*   **تصدير التقارير**: استخراج تقارير تصفية مالية بصيغة **PDF** احترافية، أو تقارير تحليلية شاملة بصيغة **Excel**.

---

## 📂 قاعدة البيانات والنسخ الاحتياطي
*   يستخدم النظام قاعدة بيانات **SQLite** محلية باسم `poultry_data.db`.
*   **ملاحظة هامة**: ملف قاعدة البيانات يتم حفظه محلياً في مجلد المشروع ولا يتم رفعه إلى GitHub لضمان خصوصية بياناتك.
*   **النسخ الاحتياطي**: يوجد زر داخل النظام لأخذ نسخة احتياطية فورية، كما يمكنك نسخ ملف `.db` يدوياً في أي وقت.

---

## 📦 متطلبات النظام (للمطورين)
إذا أردت التشغيل اليدوي دون استخدام `run.bat`:
1.  Direct run: `python main.py`
2.  المكتبات المطلوبة: `pip install -r requirements.txt`
