   نظام إدارة عنابر الدجاج اللاحم
   Poultry Farm Management System
═══════════════════════════════════════════════════════════════

متطلبات التشغيل:
──────────────────
• Python 3.8 أو أحدث (مجاني من python.org)
• مكتبة tkinter (مدمجة مع Python على Windows)
• مكتبة sqlite3 (مدمجة مع Python)
• لا يحتاج إنترنت ولا برامج إضافية

طريقة التشغيل على Windows:
─────────────────────────────
1. ثبّت Python من: https://www.python.org/downloads/
   (اختر "Add Python to PATH" أثناء التثبيت)

2. انقر مرتين على ملف "تشغيل.bat"
   أو افتح CMD واكتب:
   python main.py

قاعدة البيانات:
──────────────────
• يتم إنشاء ملف "poultry_data.db" تلقائياً في نفس المجلد
• هذا الملف يحتوي على جميع البيانات — احتفظ بنسخة منه
• يمكن فتحه وتصفحه ببرنامج DB Browser for SQLite (مجاني)
  من: https://sqlitebrowser.org/

الجداول في قاعدة البيانات:
──────────────────────────────
• warehouses  — أسماء العنابر
• batches     — جميع بيانات الدفعات (تكاليف + إيرادات + نتائج)
• v_batches   — View يجمع البيانات مع اسم العنبر

وظائف النظام:
───────────────
• إدخال دفعات جديدة مع كافة بنود التكاليف والإيرادات
• حسابات تلقائية (الأيام، إجمالي التكاليف، الإيرادات، النتيجة، النافق)
• تعديل وحذف أي دفعة
• تقرير دفعة مفردة مع توزيع التكاليف والإيرادات بالنسب
• تقرير إجمالي لكل عنبر على حدة
• تقرير شامل لجميع العنابر مرتبة حسب الأداء
• تصدير البيانات إلى CSV (يفتح في Excel)

نسخ احتياطية:
───────────────
انسخ ملف "poultry_data.db" بانتظام لحفظ بياناتك
